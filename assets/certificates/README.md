## Recommended image preparation

- Use JPG format.
- Crop unnecessary browser or phone borders.
- Keep the full certificate name, recipient name, issuer, credential/date, and certificate number visible when applicable.
- Recommended width: 1600–2400 pixels.
- Recommended file size: below 1.5 MB per image for fast loading.
- Do not upload identification documents, student records, addresses, signatures, QR codes containing private data, or unrelated personal information.

When an image is missing, the website displays the required filename instead of a broken-image icon. After the correctly named JPG is uploaded, it appears automatically and opens full-size when selected.
